# HyperBridge System Module

## Overview
This module installs **HyperBridge** as a **system application** using **KernelSU**.

- Does **not modify** the original application code  
- Does **not claim ownership** of the original app  
- Provides **system-level installation only**

The module is designed with environment checks and integrity protection to ensure safe installation.

## Installation
1. Install **KernelSU**
2. *(Optional)* Install **Mountify** if required
3. Flash the module ZIP via **KernelSU**
4. Reboot the device

If the device or environment is unsupported, installation may fail silently.
