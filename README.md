# HyperBridge System Module

## Overview
This module installs **HyperBridge** as a **system application** using **KernelSU**.
It does **not modify** the original application code and does **not claim ownership** of the app itself.

The module is intended to provide system-level installation only, with environment checks and integrity protection.

## Requirements
- **Root Method:** KernelSU only
- **Android Version:** 16 or higher
- **ROM:** Xiaomi HyperOS 3 or higher
- **Device Brand:** Xiaomi

---

## Installation
1. Install **KernelSU**
2. Flash the module ZIP via KernelSU
3. Reboot the device

If the device or environment is unsupported, installation will fail silently.
